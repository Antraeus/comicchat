# comicchat

Based off Microsoft Comic Chat.

** NOT READY FOR ANY SERIOUS USE YET: NO ESCAPING OF HTML INPUT **

## Use
1. `npm install websocket`
2. Change address of server in `client.js`
3. Change port of server in `server.js`
4. `node server.js`
5. Visit `index.html`
