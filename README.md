# comicchat

Based off Microsoft Comic Chat. Uses node.js and websockets.

![Screenshot](http://i.imgur.com/VTuJF3F.png)

## Use
0. Clone repo.
1. `npm install websocket`
2. Change address of server in `client.js`
3. Change port of server in `server.js`
4. `node server.js`
5. Visit `index.html`
